Dataset Format


The dataset should be a .csv file and the input sentences and their corresponding labels should be placed as shown in the figure below.

The dataset should not be more than 200 MB.

Allowed file extension:

.csv

(comma-separated values)

Points to note:
  • The file should have a minimum of 100 and maximum of 1000 rows.
  • You can provide as many number of classes to be classified, but make sure for per class you have atleast 100 corresponding imput sentences.
